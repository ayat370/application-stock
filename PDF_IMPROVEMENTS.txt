🔧 STRUCTURE DU CODE
================================================================================

CONSTANTES GLOBALES:
  - COLORS: Palette de couleurs cohérente
  - PAGE_MARGIN: Marge uniforme (40px)
  - LINE_HEIGHT: Hauteur de ligne standard (14px)

FONCTIONS UTILITAIRES:
  - addHeader(doc, rapport, generePar)
    → Crée en-tête professionnel avec informations
    → Retourne la position Y pour continuer après header
  
  - addFooter(doc, pageNumber)
    → Ajoute pied de page avec séparation
    → Affiche numéro de page
  
  - drawTable(doc, headers, rows, columnWidths, startY)
    → Dessine un tableau structuré avec bordures
    → Gère alternance de couleurs
    → Gère changement de page automatique
    → Retourne position finale et numéro de page

  - generatePDF(rapport, data, generePar)
    → Fonction principale de génération
    → Crée PDF selon type (inventaire/bilan)
    → Utilise toutes les fonctions utilitaires

🚀 UTILISATION
================================================================================

Les PDFs sont générés automatiquement lors de:
1. Clic sur bouton "📄 PDF" dans l'écran Rapports
2. Appel à endpoint GET /api/rapports/:id/pdf

Les PDFs contiennent:
- Métadonnées complètes (date, auteur)
- Mise en page professionnelle
- Tableaux structurés (inventaire) ou cartes (bilan)
- Numérotation des pages
- Footer avec informations
